# CT-to-MRI-Image-Translation-using-CycleGAN

## 🔹 Overview
This project implements an unsupervised image-to-image translation model using CycleGAN to convert CT images into MRI images for medical imaging applications.

## 🔹 Key Features
- Unsupervised learning using CycleGAN
- Medical image translation (CT → MRI)
- No paired dataset required
- Structural similarity optimization

## 🔹 Description
A CycleGAN-based model was developed to translate CT scan images into MRI images without requiring paired training data. The model learns mappings between the two domains using adversarial training and cycle consistency loss.

## 🔹 Results
- Achieved SSIM score of **0.58**
- Generated realistic MRI-like images from CT inputs
